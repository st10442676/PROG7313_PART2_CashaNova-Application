# CashaNova App

This project contains the CashaNova login and registration screens built with Kotlin and Jetpack Compose.

## How to open
1. Open Android Studio.
2. Select **Open**.
3. Choose the `CashaNovaAppReady` folder.
4. Wait for Gradle Sync to finish.
5. Run the app on an emulator or connected Android device.

## Included screens
- Login / welcome page
- Register page
- Navigation from login to register
- Shared background image
- Logo bundled inside the project
